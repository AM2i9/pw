    Variables and Data Types

    Variables are space in a computer's memory where a value can be stored

    Declaration

    Declares the type and name of a variable

    
							// Creates a variable called sum that should
							// store an integer
							int sum;
    <data-type> <name>;

    Primitive Data Types

  • int - an integer (1, 2, 3, 53904, 5)
  • float - a decimal number (3.14, 1.78, 5.98)
  • double - a more precise float
  • boolean - true or false
  • String - a sequence of characters, e.g. "Hello World!"
