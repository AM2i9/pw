Modulo

Modulo is division, but gets the remainder of the division after being divided equally


							1 / 1; // This is normal divison, would return 1
							1 % 1; // Would return 0, the remainder of 1/1

							6 / 4; // Would give 1.5
							6 % 4; // Would give 2, the remainder of 6 / 4

Increments/Decrements

Increments and decrements can be used to increment and decrement values


							int myVariable = 1;

							myVariable++; // myVariable now equals 2
							myVariable++; // myVariable now equals 3
							
							myVariable--; // myVaraible now equals 2

Compound Operators

IF statements

IF statements are used to only run a block of code if a certain condition is true


							if (condition) { // If 'condition' is true
								// Run the code here
							}
						

							if (condition) { // If 'condition' is true
								// Run the code here
								// Must have at least one starting if block
							} else if (otherCondition) {
								// Run if 'condition' is false, but
								// 'otherCondition' is true
								// Optional, You can have as many of these as you want
							} else {
								// Run if both conditions are false
								// Optional, Can only have one
							}
