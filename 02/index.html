Types of loops

while, for, do-while

while loops

A while loops runs while a condition is true, or until a condition is false


                        int i = 0;

                        while (i < 5) {
                            // This code will be repeated until i is not less than 5

                            i++; // Increment 'i' by 1
                            System.out.println("i is less than 5");
                            System.out.println(i);
                        }

                        System.out.println("i is now greater than or equal to 5");

do-while loops

A while loop, but the condition is checked after the code in the block is run


                        int i = 5;

                        do {
                            // This code will be repeated until i is not less than 5
                            // But, it will run at least once before checking that
                            // condition.

                            i++; // Increment 'i' by 1
                            System.out.println("i is less than 5");
                            System.out.println(i);

                        } while (i < 5);

                        System.out.println("i is now greater than or equal to 5");

for loops

Usually used to run for a specific number of repetitions


                        int i = 0;

                        while (i < 5) {
                            // This code will be repeated until i is not less than 5

                            i++; // Increment 'i' by 1
                            System.out.println("i is less than 5");
                            System.out.println(i);
                        }

                        System.out.println("i is now greater than or equal to 5");

for loops

Usually used to run for a specific number of repetitions


                        for (int i = 0; i < 5; i++) {
                            System.out.println("i is less than 5");
                            System.out.println(i);
                        }

for (initial; condition; update)

for (initial; condition; update)

initial - A statement that is executed before the start of the loop. For example, creating the variable i with int i = 0;.

condition - The condition to test for, on what condition should the loop exit? Ex. i < 5

initial - A statement that is run before every repetition of the loop, except for the first.
